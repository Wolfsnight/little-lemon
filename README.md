# Little Lemon React Project 🍋

This is a React project built with [Vite](https://vitejs.dev/), but configured to behave like a classic Create React App for simplicity.

## 🚀 Quick Start

1. **Clone the repository**

   ```bash
   git clone <REPO-URL>
   cd little-lemon
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```
   🔄 This internally uses Vite, but it's mapped to `npm start` for convenience.

## 🧱 Project Structure

- `src/components` – Reusable UI components
- `src/pages` – Pages like Home, Reservation, etc.
- `src/assets` – Images, icons, and other media
- `App.jsx` – Main application structure and routing

## 🏗 Production Build

To build the project for production:

```bash
npm run build
```

Optionally, preview the production build locally:

```bash
npm run preview
```

## ℹ️ Notes

- No additional setup required
- Runs in any modern browser
- Supports modern React features like Hooks and React Router
